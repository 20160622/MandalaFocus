# MandalaFocus

タップ・ドラッグで直観的に変化させて楽しむ、**生きている万華鏡フラクタル**のインタラクティブ・デジタルアート。

WebGL フラグメントシェーダー（GLSL）で描画し、p5.js で入力・UI を扱う、依存ライブラリは p5.js のみのシンプルな静的 Web アプリです。デスクトップのブラウザで動き、将来は Capacitor でスマホアプリ化できます。

## 起動方法

シェーダーは `fetch` で読み込むため、**ローカルサーバ経由で開く必要があります**（`index.html` をダブルクリックする `file://` では動きません）。

プロジェクト直下で、いずれかを実行してブラウザで `http://localhost:8080` を開きます。

```bash
# Python（標準で入っていることが多い）
python -m http.server 8080

# または Node.js
npx serve -l 8080
```

## 操作方法

| 操作 | 効果 |
|------|------|
| **タップ**（押してすぐ離す） | その地点から波紋が広がり成長する |
| **ドラッグ**（なぞる） | 軌跡が放射対称に複製されて描かれ、徐々に溶けていく |
| **◑ Tone** / `P` キー | 階調（グレースケールのトーン）を切り替え |
| **❖ Motif** / `M` キー | モチーフ（万華鏡・曼荼羅・法輪）を切り替え（開放済みのみ） |
| **⟳ Shuffle** / `Space` キー | 模様全体を一新 |
| **⌫ Clear** / `C` キー | 描いた線を消す |
| **⤴ Share** | 今の曼荼羅を共有コードにしてコピー |
| **📥 Load** | 共有コードを入力して再現 |

何も操作しなくても模様はゆっくり動き続けます（「生きている」アニメーション）。

## 進化の階梯（収集・進化系の要素）

タップ・ドラッグ・放置で「光」がたまり、段階的にモチーフが開放されます。
進捗は自動でブラウザに保存（localStorage）されます。

```
種 → 芽生え → 生長 → 蓮華 → 曼荼羅 → 法輪
```

- 上部HUD：現在の階梯／光の量／次の開放までの進捗バー／開放数（◉ n/6）。
- 開放されたモチーフは `❖ Motif` の巡回に加わります。
- 共有コードは seed・トーン・モチーフを 1 つの短い文字列に符号化したもの。
  `⤴ Share` でコピー、`📥 Load` で他の人の曼荼羅を再現できます。

## 構成

```
MandalaFocus/
  index.html            土台 + UIバー + HUD（p5.js を CDN 読み込み）
  style.css             全画面キャンバス・UI・HUD の見た目
  src/
    main.js             p5 sketch 本体：setup/draw・入力処理・uniform 受け渡し
    interaction.js      タップ波紋の管理 と ストローク描画バッファ
    palettes.js         グレースケール階調（トーン）と モチーフ（モード）定義
    progression.js      進化の階梯：光の蓄積・開放判定・localStorage 保存
    audio.js            静かな音：持続音(ドローン)・シンギングボウル・禅タイマーの鐘（Web Audio 合成）
  shaders/
    pattern.vert        全画面四角形を通す頂点シェーダー
    pattern.frag        中核：万華鏡フラクタル + 曼荼羅/法輪 + 等高線 + 波紋
```

## パフォーマンス調整

重い場合は `src/main.js` の `pixelDensity(1)` を下げる（例: `pixelDensity(0.75)`）か、`shaders/pattern.frag` の `fbm` のループ回数を減らします。

## 将来の展望

- **スマホアプリ化**: この静的 Web アプリを [Capacitor](https://capacitorjs.com/) でラップして iOS / Android アプリに。タッチ操作は実装済み。
- **デスクトップ単体アプリ**: 必要なら Tauri / Electron でラップ。
- **作品の保存・共有**: キャンバスの PNG 書き出し、お気に入り配色の保存 など。
